Backup

Local-first safety

Browser storage is private and convenient, but not enough for important life data. Export backups often.

This does not erase older versions because this build uses a new storage key: fixmylife-v10.2.1-state. On first launch, it can copy forward your v10.2.0 data without deleting the old key.
